# AI Agents Course: Consolidated Reference Guide

This document consolidates all topics, assignments, resources, links, and tools mentioned across the AI Agents course files. Use this as a single point of reference.

## 📅 Learning Path by Topic

### 1. Introduction & Vibe Coding
* **Theme**: Transitioning from manual syntax coding to intent-driven "vibe coding" and disciplined "agentic engineering."
* **Core Concepts**: Traditional SDLC compression, the "factory model" (developers as system orchestrators designing evaluations, constraints, and context harnesses).
* **Assignments & Media**:
  * 🎧 **Podcast**: [Summary Podcast (Unit 1)](https://www.youtube.com/watch?v=cbzmr7vt4XA)
  * 📄 **Whitepaper**: [The New SDLC with Vibe Coding](https://www.kaggle.com/whitepaper-the-new-SDLC-with-vibe-coding)
  * 💻 **Codelabs**:
    1. [Get started with Antigravity 2.0 and IDE](https://codelabs.developers.google.com/getting-started-google-antigravity#0)
    2. [Build a Web Application in AI Studio and Deploy to Cloud Run](https://codelabs.developers.google.com/deploy-from-aistudio-to-run?hl=en#0)
  * 🎥 **Livestream Recording**: [Unit 1 Livestream](https://youtube.com/live/7iic3Zj427M)

### 2. Agent Tools & MCP
* **Theme**: Standardizing a plug-and-play AI ecosystem using open protocols.
* **Core Concepts**: Model Context Protocol (MCP) for connecting models to data, Agent-to-Agent (A2A) collaboration, Agent-to-User Interface (A2UI) for generative UI, Agent Payments Protocol (AP2), and Universal Commerce Protocol (UCP).
* **Assignments & Media**:
  * 🎧 **Podcast**: [Summary Podcast (Unit 2)](https://www.youtube.com/watch?v=GjjKXqxFTOY)
  * 📄 **Whitepaper**: [Agent Tools & Interoperability](https://www.kaggle.com/whitepaper-agent-tools-and-interoperability)
  * 💻 **Codelabs**:
    1. [Get started with Antigravity CLI](https://codelabs.developers.google.com/antigravity-cli-hands-on#0)
    2. [Explore Google Developer Knowledge MCP server in Google Antigravity](https://codelabs.developers.google.com/developer-knowledge-mcp-antigravity)
  * 🎥 **Livestream Recording**: [Unit 2 Livestream](https://youtube.com/live/PGI_S59EoRA)

### 3. Agent Skills
* **Theme**: Managing dynamic context, reducing "context rot", and implementing reusable agent capabilities.
* **Core Concepts**: Portable "Agent Skills" directories structured around a central `SKILL.md` file, progressive disclosure to keep system prompts lightweight, loading tools/instructions on demand.
* **Assignments & Media**:
  * 🎧 **Podcast**: [Summary Podcast (Unit 3)](https://www.youtube.com/watch?v=uYURYHhpmKc)
  * 📄 **Whitepaper**: [Agent Skills](https://www.kaggle.com/whitepaper-agent-skills)
  * 💻 **Codelabs**:
    1. [Explore how Skills work in Antigravity](https://codelabs.developers.google.com/getting-started-with-antigravity-skills?hl=en#4)
    2. [Build agents in Antigravity with Agents CLI and ADK](https://codelabs.developers.google.com/agents-cli-adk-lifecycle)
  * 🎥 **Livestream Recording**: [Unit 3 Livestream](https://youtube.com/live/1T2mxYZkqL0)

### 4. Security & Evaluation
* **Theme**: Redefining evaluation and security standards in non-deterministic workflows.
* **Core Concepts**: Continuous "Effective Trust" via a strict 7-pillar architecture, ephemeral sandboxing, defenses against hallucinated package names ("slopsquatting"), Red/Blue/Green security triad, and OpenTelemetry trajectory evaluation.
* **Assignments & Media**:
  * 🎧 **Podcast**: [Summary Podcast (Unit 4)](https://www.youtube.com/watch?v=Ddz1b8CYPvg)
  * 📄 **Whitepaper**: [Vibe Coding Agent Security and Evaluation](https://www.kaggle.com/whitepaper-vibe-coding-agent-security-and-evaluation)
  * 💻 **Codelabs**:
    1. [Build an expense-approval agent with human-in-the-loop triage and local evaluations with ADK, Agents CLI and Antigravity](https://codelabs.developers.google.com/vibecode-ambient-expense-agent)
    2. [Write Secure AI Code: Automated Threat Scans, Safety Guards, and Security Testing](https://codelabs.developers.google.com/secure-agentic-coding)
  * 🎥 **Livestream Recording**: [Unit 4 Livestream](https://www.youtube.com/live/suWoYLD7uGY)

### 5. Production & Scaling
* **Theme**: Transitioning vibe-coded prototypes to scalable, governed, observable enterprise fleets.
* **Core Concepts**: Spec-Driven Development (SDD), treating code as disposable, behavior-driven Gherkin specifications as the source of truth, zero-trust deployment pipelines, automated code-review agents, hybrid Policy Servers.
* **Assignments & Media**:
  * 🎧 **Podcast**: [Summary Podcast (Unit 5)](https://www.youtube.com/watch?v=VSRdL4wlbLY)
  * 📄 **Whitepaper**: [Spec-Driven Production Grade Development in the Age of Vibe Coding](https://www.kaggle.com/whitepaper-spec-driven-production-grade-development-in-the-age-of-vibe-coding)
  * 💻 **Codelabs**:
    1. [Deploy and host your AI agents on Google Cloud](https://codelabs.developers.google.com/enterprise-cloud-scale-deploying-the-expense-agent-to-agent-runtime-on-google-cloud)
    2. [Build a front-end web app and link it to your cloud-hosted AI agent](https://codelabs.developers.google.com/vibecode-frontend-with-antigravity)
  * 🎥 **Livestream Recording**: [Unit 5 Livestream](https://youtube.com/live/Y3HfV4IroCU)
